# app-gen

A flexible and extensible Web Application Framework that lets you reclaim ownership of your code whenever you need.

## Concept

Write the domain logic with responsibility, and let the coding agent build the rest of your Web Application.

Unlike traditional frameworks, all generated code is readable and editable. You're never locked inâ€”take full control of your code at any time.

## Skills

| Skill | Description |
|-------|-------------|
| `analyze-relations` | Analyze relationships between entities from domain models |
| `analyze-layers` | Derive layer structure from application requirements |
| `design-structure` | Design directory structure from layer structure |
| `webapp` | Generate Web Application code from domain models |
